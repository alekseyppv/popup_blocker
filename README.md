# Popup Blocker (Chrome Extension)

Расширение блокирует все новые вкладки/окна на выбранных сайтах, кроме вкладок, открытых средней кнопкой мыши.

## Что делает

- Блокирует новые окна и всплывающие вкладки, открытые обычным кликом, скриптами, `window.open`, `target="_blank"` и т.д. на доменах из списка.
- Разрешает открытие только вкладок из текущего окна, если пользователь открыл ссылку **средней кнопкой мыши**.
- Позволяет управлять списком доменов через popup-меню расширения.
- Позволяет добавить домен текущей активной вкладки одной кнопкой.

## Установка в Chrome

1. Откройте `chrome://extensions/`.
2. Включите **Developer mode**.
3. Нажмите **Load unpacked**.
4. Выберите папку проекта (`/workspace/popup_blocker`).

## Использование

1. Откройте popup расширения.
2. Нажмите **Добавить текущий домен** для сайта, где нужно блокировать рекламные окна.
3. При необходимости удаляйте домены из списка кнопкой **Удалить**.
